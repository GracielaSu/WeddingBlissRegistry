Make sure PHP has been installed on your computer

Step 1: Run xampp-control
Step 2: Under Apache, click "Config" > "Apache (httpd.conf)"
Step 3: In the file that opens, replace SRVROOT and ServerRoot with the path that you have saved the folder to, including /apache at the end
Step 4: Replace DocumentRoot and <Directory> with the path that you have saved the folder to, including /htdocs/WeddingProject at the end
Step 5: Start Apache and MySQL services within XAMPP Control Panel
Step 6: Visit http://locahost to view the website
